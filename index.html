public beta · privacy proxy for AI coding agents

Stop leaking secrets to your AI.

NoviSentinel is the open-source layer between your coding agent and the LLM. It detects secrets and PII, swaps them for stable placeholders before the request leaves your machine, and restores the originals in the response — so your agent keeps working, while your AWS keys never touch a third-party server.

Get started View on GitHub apache 2.0 · self-hosted
01 — WHY IT EXISTS

The new exfiltration vector is the chat box.

Every dev has pasted a code snippet with a secret in it into Claude or ChatGPT. That key now lives in a third-party log forever. NoviSentinel makes that habit safe — your secrets never leave your machine, but your agent still gets the work done.

01

Local-first.

Runs entirely on your machine. No SaaS dashboard, no telemetry, no master key. The mapping between your secrets and their placeholders lives in memory for one request, then dies.

  • Single docker run
  • No outbound calls besides your LLM provider
  • Apache 2.0, audit the code yourself
02

10+ secret types, plus PII.

Regex detection for the obvious ones — AWS, OpenAI, Anthropic, GitHub, Slack, Stripe, JWT. Presidio + spaCy for soft PII — emails, phones, SSNs, credit cards, IBANs, IPs.

  • Critical secrets blocked; PII redacted-and-forwarded
  • Stable placeholders preserve coreference
  • Custom detectors via HTTP endpoint
03

Drop-in for any agent.

Speaks both /v1/chat/completions and /v1/messages. Set one env var on your agent and the proxy is invisible — Claude Code, Cursor, Aider, Codex, Cline, raw SDK calls, all work.

  • Anthropic, OpenAI, and Google Gemini upstreams
  • Streaming preserved end-to-end
  • Prompt-injection detection on the way in
02 — HOW IT WORKS

Redact on the way out, restore on the way back.

Each request runs through seven parallel detectors. Anything privacy-sensitive is swapped for a stable placeholder — same value gets the same token throughout the request, so the model can still reason about context. Placeholders are reversed transparently in the response.

04 — INSTALL

One command. Pick your surface.

The Docker image is the canonical install — every dependency is baked in (Presidio, spaCy model, the injection classifier, Detoxify). The Python SDK is for programmatic use. The VS Code extension is for inline scan-as-you-type.

Docker

canonical · self-hosted

# pull and run
docker run -d -p 8000:8000 \
  ghcr.io/009-kumarji/novisentinel:latest

# point your agent
export OPENAI_BASE_URL=http://localhost:8000/v1
export ANTHROPIC_BASE_URL=http://localhost:8000

Python SDK

pypi · programmatic scans

# install
pip install novisentinel

# use
from novisentinel import Client
client = Client()
result = client.scan("my email is alice@acme.com")
print(result.detections)
